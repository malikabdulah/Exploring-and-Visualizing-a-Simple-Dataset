# Exploring and Visualizing a Simple Dataset

**Organization:** DevelopersHub Corporation  
**Domain:** Artificial Intelligence & Machine Learning  

## Objective
The goal of this task is to learn how to load, inspect, and visualize a dataset to uncover data trends and distributions. This repository contains the Exploratory Data Analysis (EDA) performed on the classic **Iris Dataset**.

## Tech Stack & Libraries
* **Language:** Python
* **Data Manipulation:** Pandas
* **Data Visualization:** Matplotlib, Seaborn

## Key Skills Demonstrated
* **Data Loading & Inspection:** Utilizing `pandas` to read datasets and inspect shapes, columns, and head rows.
* **Descriptive Statistics:** Leveraging `.info()` and `.describe()` to identify data types, missing values, and mathematical summaries.
* **Data Visualization:** * **Scatter Plots:** To map relationships between features.
  * **Histograms:** To visualize the frequency distribution of continuous variables.
  * **Box Plots:** To identify statistical outliers within the data.
